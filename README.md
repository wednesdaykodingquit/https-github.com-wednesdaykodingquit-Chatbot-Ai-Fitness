# AI-Powered Personal Fitness Companion

This project focuses on developing an AI-powered personal fitness companion that operates locally without relying on a database. The application leverages Generative AI models to provide personalized fitness plans, diet recommendations, and a fitness chatbot for real-time guidance. The core functionality includes BMI-based fitness analysis, workout plan generation, diet and nutrition planning, and a chatbot powered by LLM APIs. The application will dynamically generate outputs based on user inputs and display them instantly through an interactive user interface. This project emphasizes frontend-backend integration with AI, prompt engineering, and local development using VS Code.
